# Netflix-Movies-TV-Shows-Data-Cleaning-Excel
This project focuses on cleaning and preparing a real-world Netflix Movies and TV Shows dataset using Microsoft Excel. The dataset contained missing values, duplicates, and inconsistent formats which were cleaned to make it analysis-ready.
- Collected and worked on a real-world Netflix Movies & TV Shows dataset in CSV format
- Performed data cleaning using Microsoft Excel
- Identified and replaced missing (blank) values
- Checked and handled duplicate records
- Standardized text data using TRIM and PROPER functions
- Validated and corrected date and categorical data formats
- Created a separate Cleaned_Data sheet to keep raw and cleaned data separate
- Added Data_Quality_Notes to document data quality issues
- Exported the final cleaned dataset into Excel and CSV formats
